# 3D Machine

## IMPORTANT
This Cell Machine "remake" uses alot of code from CMMM, so it has the same license as CMMM and it is considered a mod of CMMM


## Controls
~~~
e/q      rotate
z/x      switch cells
w/a/s/d  moving
mouse    rotate camera
esc      open the menu
r        play/pause the simulation
~~~

## Cells
~~~
Mover
Trash
Immobile
Fixed Rotator
~~~
